项目结构：
---css  css,less文件
---src
    --asset 图片资源
    --api   数据接口
    --store vuex状态管理
    --utils 工具文件
    --route 路由文件
    --pages 页面
    --components 组件
        -album 专辑相关
        -comment 评论
        -common  公共组件
        -control  播放控制音乐组件
        -nav    头部导航组件
        -newDishes 新碟组件
        -playList  歌单相关
        -rankComp  榜单相关
        -songComp  歌曲相关
